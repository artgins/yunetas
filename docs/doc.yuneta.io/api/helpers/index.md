# Helpers API

Standalone utility functions used throughout the kernel: string and JSON
manipulation, JSON-path addressing, time/date conversion, file-system access,
buffers, lists, memory, and more.

## In this section

- [String](string_helper.md) — string utilities.
- [JSON](json_helper.md) — JSON construction and traversal helpers.
- [kwid (JSON path)](kwid.md) — addressing values inside JSON trees.
- [Time & Date](time_date.md) — timestamp and date conversion.
- [File System](file_system.md) — file and path helpers.
- [Directory Walk](directory_walk.md) — recursive directory traversal.
- [URL Parsing](url_parsing.md) — URL splitting and composition.
- [HTTP Parser](http_parser.md) — the HTTP message parser wrapper.
- [IStream](istream.md) — incremental stream reading.
- [GBuffer](gbuffer.md) — the growable byte buffer.
- [Doubly-Linked List](dl.md) — the intrusive list implementation.
- [Memory](memory.md) — the `gbmem_*` allocator wrappers.
- [Common Protocol](common_protocol.md) — shared protocol helpers.
- [Daemon Launcher](daemon_launcher.md) — daemonising a process.
- [Backtrace](backtrace.md) — stack-trace capture.
- [Misc](misc.md) — everything else.
