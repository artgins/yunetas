# `get_number_of_files()`
<!-- ============================================================== -->


Get the number of files in a specified directory.

``````{tab-set}
    const char  *path

```

**Parameters**


:::{list-table}
:widths: 10 5 40
:header-rows: 1
* - Key
  - Type
  - Description

* - `path`
``````
