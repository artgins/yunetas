# `is_regular_file()`
<!-- ============================================================== -->


Check if the specified path points to a regular file.

:::{list-table}
:widths: 10 5 40
:header-rows: 1
* - Key
  - Type
  - Description

* - `path`
  - `const char *`

**Return Value**


Returns `TRUE` if the path points to a regular file, otherwise returns `FALSE`.
