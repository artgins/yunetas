# `trq_size()`
<!-- ============================================================== -->

`trq_size()` returns the number of messages currently stored in the specified queue.

<!------------------------------------------------------------>
<!--                    Prototypes                          -->
<!------------------------------------------------------------>

``````{tab-set}

`````{tab-item} C

<!--====================================================-->
<!--                    Tab C                           -->
<!--====================================================-->

**Prototype**

```C
size_t trq_size(
    tr_queue trq
);
```

**Parameters**

::: {list-table}
:widths: 20 20 60
:header-rows: 1

* - Key
  - Type
  - Description

* - `trq`
  - `tr_queue`
  - The queue whose size is to be determined.
:::

---

**Return Value**

The number of messages in the queue.

**Notes**

This function provides a quick way to check the number of messages stored in a queue.
``````
