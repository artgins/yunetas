# `tm2timestamp()`
<!-- ============================================================== -->


Convert a `struct tm` to a timestamp in seconds since the epoch.

<!--                    Prototypes                          -->
<!------------------------------------------------------------>

``````{tab-set}

`````{tab-item} C

<!--====================================================-->
<!--                    Tab C                           -->
<!--====================================================-->

**Prototype**

```C

PUBLIC double tm2timestamp(
    const struct tm *tm
);
        

```

**Parameters**


:::{list-table}
:widths: 20 20 60
:header-rows: 1
* - Key
  - Type
  - Description

* - `tm`
  - `const struct tm *`
  - A pointer to the `struct tm` to convert.
:::
        

---

**Return Value**


Returns the timestamp as a `double` value in seconds since the epoch.
``````
