(trq_topic())=
# `trq_topic()`
<!-- ============================================================== -->

`trq_topic()` returns the topic associated with the given queue.

<!------------------------------------------------------------>
<!--                    Prototypes                          -->
<!------------------------------------------------------------>

``````{tab-set}

`````{tab-item} C

<!--====================================================-->
<!--                    Tab C                           -->
<!--====================================================-->

**Prototype**

```C
json_t *trq_topic(
    tr_queue trq
);
```

**Parameters**

::: {list-table}
:widths: 20 20 60
:header-rows: 1

* - Key
  - Type
  - Description

* - `trq`
  - `tr_queue`
  - The queue instance from which to retrieve the topic.
:::

---

**Return Value**

A [`json_t *`](#json_t) representing the topic of the queue.

**Notes**

The returned [`json_t *`](#json_t) is owned by the queue and should not be modified or freed by the caller.
``````
