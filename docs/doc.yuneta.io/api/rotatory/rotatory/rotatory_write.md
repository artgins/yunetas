# `rotatory_write()`
<!-- ============================================================== -->


Write data to a rotatory log file.
        

<!------------------------------------------------------------>
<!--                    Prototypes                          -->
<!------------------------------------------------------------>

``````{tab-set}

`````{tab-item} C

<!--====================================================-->
<!--                    Tab C                           -->
<!--====================================================-->

**Prototype**

```C

PUBLIC int rotatory_write(
    int          handle,
    const void  *data,
    size_t       size
);
        

```

**Parameters**


:::{list-table}
:widths: 20 20 60
:header-rows: 1
* - Key
  - Type
  - Description

* - `handle`
  - `int`
  - The handle of the rotatory log to write to.

* - `data`
  - `const void *`
  - The data to write.

* - `size`
  - `size_t`
  - The size of the data in bytes.
:::
        

---

**Return Value**


Returns the number of bytes successfully written, or a negative value on failure.
``````
