(trq_load())=
# `trq_load()`
<!-- ============================================================== -->

`trq_load()` loads pending messages from the queue and returns an iterator for traversal.

<!------------------------------------------------------------>
<!--                    Prototypes                          -->
<!------------------------------------------------------------>

``````{tab-set}

`````{tab-item} C

<!--====================================================-->
<!--                    Tab C                           -->
<!--====================================================-->

**Prototype**

```C
int trq_load(
    tr_queue trq
);
```

**Parameters**

::: {list-table}
:widths: 20 20 60
:header-rows: 1

* - Key
  - Type
  - Description

* - `trq`
  - `tr_queue`
  - The queue instance from which pending messages will be loaded.
:::

---

**Return Value**

Returns an iterator for traversing the loaded messages.

**Notes**

Use [`trq_load_all()`](#trq_load_all()) to load all messages, including non-pending ones.
``````
