# `trq_check_backup()`
<!-- ============================================================== -->

`trq_check_backup()` performs a backup operation on the queue if necessary.

<!------------------------------------------------------------>
<!--                    Prototypes                          -->
<!------------------------------------------------------------>

``````{tab-set}

`````{tab-item} C

<!--====================================================-->
<!--                    Tab C                           -->
<!--====================================================-->

**Prototype**

```C
int trq_check_backup(
    tr_queue trq
);
```

**Parameters**

::: {list-table}
:widths: 20 20 60
:header-rows: 1

* - Key
  - Type
  - Description

* - `trq`
  - `tr_queue`
  - The queue instance to check and perform backup if needed.
:::

---

**Return Value**

Returns `0` on success, or a negative value if an error occurs.

**Notes**

This function ensures that the queue's backup mechanism is triggered when required.
``````
