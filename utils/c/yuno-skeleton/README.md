# yuno-skeleton

Create a new yuno from a pre-defined template. Lists the available skeletons or instantiates one into the current directory with the target name and optional path.

## Usage

```bash
yuno-skeleton --list                           # show available templates
yuno-skeleton <template> <name> [<path>]       # instantiate
```

See also `ymake-skeleton` for cloning from an existing project rather than a template library.
