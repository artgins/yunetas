# `gobj_get_deep_tracing()`
<!-- ============================================================== -->


Check if deep tracing is enabled for a GObj.

<!--                    Prototypes                          -->

``````{tab-set}
```

**Parameters**


:::{list-table}
:widths: 20 20 60
:header-rows: 1
* - Key
  - Type
  - Description

* - `gobj`
  - `hgobj`
  - The GObj to check for deep tracing.

**Return Value**


Returns `TRUE` if deep tracing is enabled, otherwise returns `FALSE`.
``````
