# `gobj_get_gclass_trace_level_list()`
<!-- ============================================================== -->


Retrieve a list of all enabled trace levels for a GClass.

**Parameters**


:::{list-table}
:widths: 20 20 60
:header-rows: 1
* - Key
  - Type
  - Description

* - `gclass`
