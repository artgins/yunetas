(json_record_to_schema())=
# `json_record_to_schema()`
<!-- ============================================================== -->


Converts a JSON record description to a topic schema. The function takes a JSON description of a record structure and converts it into a schema format suitable for topics.


<!------------------------------------------------------------>
<!--                    Prototypes                          -->
<!------------------------------------------------------------>

``````{tab-set}

`````{tab-item} C

<!--====================================================-->
<!--                    Tab C                           -->
<!--====================================================-->

**Prototype**

```C

json_t *json_record_to_schema(
    const json_desc_t *json_desc
);

```

**Parameters**


::: {list-table}
:widths: 20 20 60
:header-rows: 1

* - Key
  - Type
  - Description

* - `json_desc`
  - `const json_desc_t *`
  - JSON description of the record structure.

:::


---

**Return Value**


Returns a JSON object representing the schema of the record structure.


**Notes**


- The `json_desc_t` structure defines the fields of the record with attributes like name, type, default value, and fillspace.
- The schema generated by this function is suitable for defining topics or data structures based on the JSON record description.
``````
