# `tranger2_shutdown()`
<!-- ============================================================== -->

The `tranger2_shutdown()` function shuts down the TimeRanger database, releasing all allocated memory.

<!------------------------------------------------------------>
<!--                    Prototypes                          -->
<!------------------------------------------------------------>

``````{tab-set}

`````{tab-item} C

<!--====================================================-->
<!--                    Tab C                           -->
<!--====================================================-->

**Prototype**

```C
int tranger2_shutdown(
    json_t *tranger
);
```

**Parameters**

::: {list-table}
:widths: 20 20 60
:header-rows: 1

* - Key
  - Type
  - Description

* - `tranger`
  - `json_t *`
  - A pointer to the TimeRanger database instance to be shut down.
:::

---

**Return Value**

Returns `0` on success, or a negative value if an error occurs.

**Notes**

This function should be called when the database is no longer needed to free resources.
``````
