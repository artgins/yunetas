# `gobj_log_debug()`
<!-- ============================================================== -->

Logs a debug-level message with formatting options.

```C

PUBLIC void gobj_log_debug(hgobj gobj, log_opt_t opt, ...);
        

```

**Parameters**


:::{list-table}
:widths: 20 20 60
:header-rows: 1
* - Key
  - Type
  - Description

* - `gobj`
  - [`hgobj`](hgobj)
  - Handle to the GObj instance.

* - `opt`
  - `log_opt_t`
  - Log options (e.g., stack trace or additional metadata).

* - `...`
  - `variadic`
