# Timeranger2 API

The append-only time-series persistence engine and the layers built on top of
it: the message store, the dict-style msg2db, persistent queues, the treedb
graph database, and the file-system watcher.

## In this section

- [timeranger2](timeranger2.md) — the core append-only, key-indexed store.
- [tr_msg](tr_msg.md) — the message store.
- [tr_msg2db](tr_msg2db.md) — the dict-style message database.
- [tr_queue](tr_queue.md) — persistent message queues.
- [tr_treedb](treedb.md) — the graph (nodes/links) database.
- [fs_watcher](fs_watcher.md) — the filesystem-watcher library API.
