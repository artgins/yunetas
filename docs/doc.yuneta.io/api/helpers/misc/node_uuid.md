# `node_uuid()`
<!-- ============================================================== -->


This function retrieves the universally unique identifier (UUID) of the machine on which the code is running. 
The UUID is a 128-bit number used to uniquely identify the machine in a distributed system. 
This function does not take any parameters and returns a pointer to a string representation of the UUID.

**Return Value**


The function returns a pointer to a string that represents the UUID of the machine. 
This string is dynamically allocated and should be freed by the caller when no longer needed.


**Notes**


The UUID generated by this function is unique to the machine and can be used for identification purposes in various applications. 
It is important to ensure that the returned string is properly managed to avoid memory leaks.
