# `yev_get_state_name()`

## Description
Retrieves the state name of an event.

## Prototype
```c
PUBLIC const char *yev_get_state_name(yev_event_h yev_event);
```

## Parameters
- `yev_event` (`yev_event_h`): The event handle.

## Return Value
- Returns a string representing the state name of the event.
