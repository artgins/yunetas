# `tranger2_stop()`
<!-- ============================================================== -->

The `tranger2_stop()` function closes the TimeRanger database, ensuring that all topics and file descriptors are properly closed.

<!------------------------------------------------------------>
<!--                    Prototypes                          -->
<!------------------------------------------------------------>

``````{tab-set}

`````{tab-item} C

<!--====================================================-->
<!--                    Tab C                           -->
<!--====================================================-->

**Prototype**

```C
int tranger2_stop(
    json_t *tranger
);
```

**Parameters**

::: {list-table}
:widths: 20 20 60
:header-rows: 1

* - Key
  - Type
  - Description

* - `tranger`
  - `json_t *`
  - A pointer to the TimeRanger database instance to be closed.
:::

---

**Return Value**

Returns `0` on success, or a negative value if an error occurs.

**Notes**

This function should be called before shutting down the database using [`tranger2_shutdown()`](#tranger2_shutdown()).
``````
