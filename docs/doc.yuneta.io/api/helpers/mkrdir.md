(mkrdir())=
# `mkrdir()`
<!-- ============================================================== -->


Create a directory and all its parent directories if they do not exist.
        

<!------------------------------------------------------------>
<!--                    Prototypes                          -->
<!------------------------------------------------------------>

``````{tab-set}

`````{tab-item} C

<!--====================================================-->
<!--                    Tab C                           -->
<!--====================================================-->

**Prototype**

```C

PUBLIC int mkrdir(
    const char  *path,
    int         mode
);
        

```

**Parameters**


:::{list-table}
:widths: 10 5 40
:header-rows: 1
* - Key
  - Type
  - Description

* - `path`
  - `const char *`
  - The path of the directory to create.

* - `mode`
  - `int`
  - The permissions mode for the directory.
:::
        

---

**Return Value**


Returns `0` on success, or a negative value on failure.
``````
