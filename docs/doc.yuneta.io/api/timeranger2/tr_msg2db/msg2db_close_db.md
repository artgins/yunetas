# `msg2db_close_db()`
<!-- ============================================================== -->

`msg2db_close_db()` closes an open message database identified by `msg2db_name`, ensuring that all resources associated with it are properly released.

<!------------------------------------------------------------>
<!--                    Prototypes                          -->
<!------------------------------------------------------------>

``````{tab-set}

`````{tab-item} C

<!--====================================================-->
<!--                    Tab C                           -->
<!--====================================================-->

**Prototype**

```C
int msg2db_close_db(
    json_t  *tranger,
    const char *msg2db_name
);
```

**Parameters**

::: {list-table}
:widths: 20 20 60
:header-rows: 1

* - Key
  - Type
  - Description

* - `tranger`
  - `json_t *`
  - A reference to the TimeRanger database instance.

* - `msg2db_name`
  - `const char *`
  - The name of the message database to close.
:::

---

**Return Value**

Returns `0` on success, or a negative value if an error occurs.

**Notes**

Ensure that [`msg2db_close_db()`](#msg2db_close_db()) is called after all operations on the database are complete to prevent resource leaks.
``````
