# `trq_msg_md_record()`

**Description:**
Retrieves the metadata record associated with a message.

**Prototype:**
```c
PUBLIC md2_record_ex_t *trq_msg_md_record(q_msg *msg);
```

**Parameters:**
- `msg` (`q_msg *`): The message instance.

**Returns:**
- `md2_record_ex_t *`: The metadata record of the message.
