# `split_free3()`
<!-- ============================================================== -->


The `split_free3` function is used to free the memory allocated for a list of strings generated by the `split3` function.


<!------------------------------------------------------------>
<!--                    Prototypes                          -->
<!------------------------------------------------------------>

``````{tab-set}

`````{tab-item} C

<!--====================================================-->
<!--                    Tab C                           -->
<!--====================================================-->

**Prototype**

```C

void split_free3(
    const char **list
);

```

**Parameters**


::: {list-table}
:widths: 20 20 60
:header-rows: 1

* - Key
  - Type
  - Description

* - `list`
  - `const char **`
  - The list of strings to be freed.
:::


---

**Return Value**


This function does not return any value.


**Notes**


- The `split3` function is used to split a string by delimiter characters, returning a list of strings that may include empty strings.
- It is important to use `split_free3` to deallocate the memory allocated for the list of strings to prevent memory leaks.
``````
