# README

This is an adaptation of libjwt https://github.com/benmcollins/libjwt to Yuneta.

Previous version of Yuneta was working well with the version libjwt-1.16.0,
but changes done in version 2 and 3 will get incompatible with Yunetas,
having the need to integrate libjwt directly in Yunetas with the required modifications.
