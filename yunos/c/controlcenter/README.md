# controlcenter

Central management console (yuno) for distributed Yuneta systems. Connects to running yunos, shows real-time status, forwards events, and provides an interactive control surface for administrators.

Pairs with `yuno_agent22` (the alternate backdoor agent) for remote management.

## Build / run

Built as part of the `yunos/c/` tree. See top-level `CLAUDE.md` and `doc.yuneta.io` for deployment and configuration.
