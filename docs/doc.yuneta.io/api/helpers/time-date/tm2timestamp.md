(tm2timestamp())=
# `tm2timestamp()`
<!-- ============================================================== -->

Converts a `struct tm` time representation into an ISO 8601 formatted timestamp string.

<!------------------------------------------------------------>
<!--                    Prototypes                          -->
<!------------------------------------------------------------>

``````{tab-set}

`````{tab-item} C

<!--====================================================-->
<!--                    Tab C                           -->
<!--====================================================-->

**Prototype**

```C
char *tm2timestamp(
    char *bf,
    int   bfsize,
    struct tm *tm
);
```

**Parameters**

::: {list-table}
:widths: 20 20 60
:header-rows: 1

* - Key
  - Type
  - Description

* - `bf`
  - `char *`
  - Buffer to store the formatted timestamp.

* - `bfsize`
  - `int`
  - Size of the buffer `bf`.

* - `tm`
  - `struct tm *`
  - Pointer to a `struct tm` containing the time to be formatted.
:::

---

**Return Value**

Returns a pointer to the buffer `bf` containing the formatted timestamp.

**Notes**

The output format follows the ISO 8601 standard: `YYYY-MM-DDTHH:MM:SS.0Â±HHMM`.
``````
