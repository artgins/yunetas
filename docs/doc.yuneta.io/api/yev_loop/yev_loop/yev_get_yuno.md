(yev_get_yuno())=
# `yev_get_yuno()`
<!-- ============================================================== -->

`yev_get_yuno()` retrieves the `yuno` object associated with the given event loop.

<!------------------------------------------------------------>
<!--                    Prototypes                          -->
<!------------------------------------------------------------>

``````{tab-set}

`````{tab-item} C

<!--====================================================-->
<!--                    Tab C                           -->
<!--====================================================-->

**Prototype**

```C
hgobj yev_get_yuno(
    yev_loop_h yev_loop
);
```

**Parameters**

::: {list-table}
:widths: 20 20 60
:header-rows: 1

* - Key
  - Type
  - Description

* - `yev_loop`
  - `yev_loop_h`
  - Handle to the event loop from which to retrieve the `yuno` object.
:::

---

**Return Value**

Returns the [`hgobj`](#hgobj) object associated with the specified event loop.

**Notes**

The returned [`hgobj`](#hgobj) may be `NULL` if the event loop is not properly initialized.
``````
