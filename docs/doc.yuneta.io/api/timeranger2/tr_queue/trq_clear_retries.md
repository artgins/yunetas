# `trq_clear_retries()`
<!-- ============================================================== -->

Clears the retry count of the specified message in the queue.

<!------------------------------------------------------------>
<!--                    Prototypes                          -->
<!------------------------------------------------------------>

``````{tab-set}

`````{tab-item} C

<!--====================================================-->
<!--                    Tab C                           -->
<!--====================================================-->

**Prototype**

```C
void trq_clear_retries(
    q_msg msg
);
```

**Parameters**

::: {list-table}
:widths: 20 20 60
:header-rows: 1

* - Key
  - Type
  - Description

* - `msg`
  - `q_msg`
  - The message whose retry count should be cleared.
:::

---

**Return Value**

This function does not return a value.

**Notes**

Use [`trq_test_retries()`](#trq_test_retries()) to check the retry count before clearing it.
``````
