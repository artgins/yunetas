# `filesize2()`
<!-- ============================================================== -->


Get the size of a file in bytes using a file descriptor.
        

<!------------------------------------------------------------>
<!--                    Prototypes                          -->
<!------------------------------------------------------------>

``````{tab-set}

`````{tab-item} C

<!--====================================================-->
<!--                    Tab C                           -->
<!--====================================================-->

**Prototype**

```C

PUBLIC off_t filesize2(
    int         fd
);
        

```

**Parameters**


:::{list-table}
:widths: 20 20 60
:header-rows: 1
* - Key
  - Type
  - Description

* - `fd`
  - `int`
  - The file descriptor of the file to query.
:::
        

---

**Return Value**


Returns the size of the file in bytes, or a negative value on failure.
``````
