(dl_next())=
# `dl_next()`
<!-- ============================================================== -->


Get the next element in the doubly-linked list relative to a given node.
        

<!------------------------------------------------------------>
<!--                    Prototypes                          -->
<!------------------------------------------------------------>

``````{tab-set}

`````{tab-item} C

<!--====================================================-->
<!--                    Tab C                           -->
<!--====================================================-->

**Prototype**

```C

PUBLIC dl_node_t *dl_next(
    dl_node_t   *node
);
        

```

**Parameters**


:::{list-table}
:widths: 20 20 60
:header-rows: 1
* - Key
  - Type
  - Description

* - `node`
  - [`dl_node_t *`](dl_node_t)
  - The current node.
:::
        

---

**Return Value**


Returns a pointer to the next node, or `NULL` if there is no next node.
``````
