# `yev_start_event()`
<!-- ============================================================== -->

`yev_start_event()` starts the specified event, transitioning it to the running state if applicable.

<!------------------------------------------------------------>
<!--                    Prototypes                          -->
<!------------------------------------------------------------>

``````{tab-set}

`````{tab-item} C

<!--====================================================-->
<!--                    Tab C                           -->
<!--====================================================-->

**Prototype**

```C
int yev_start_event(
    yev_event_h yev_event
);
```

**Parameters**

::: {list-table}
:widths: 20 20 60
:header-rows: 1

* - Key
  - Type
  - Description

* - `yev_event`
  - `yev_event_h`
  - Handle to the event that should be started.
:::

---

**Return Value**

Returns 0 on success, or a negative value on failure.

**Notes**

For timer events, use [`yev_start_timer_event()`](#yev_start_timer_event()) instead.
``````
