# stats_list

List persisted statistics from a Yuneta stats store. Supports filtering by group, variable name, metric, units, and time range, with optional recursive traversal and raw output.

## Usage

```bash
stats_list [--path <dir>] [--group <g>] [--variable <v>] [--from <t>] [--to <t>] [options]
```

Run `stats_list --help` for the complete option set.
