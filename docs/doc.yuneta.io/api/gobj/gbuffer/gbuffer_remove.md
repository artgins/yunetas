# `gbuffer_remove()`
<!-- ============================================================== -->


Remove and free a gbuffer from memory.
        

<!------------------------------------------------------------>
<!--                    Prototypes                          -->
<!------------------------------------------------------------>

``````{tab-set}

`````{tab-item} C

<!--====================================================-->
<!--                    Tab C                           -->
<!--====================================================-->

**Prototype**

```C

PUBLIC int gbuffer_remove(
    gbuffer_t   *gbuffer
);
        

```

**Parameters**


:::{list-table}
:widths: 20 20 60
:header-rows: 1
* - Key
  - Type
  - Description

* - `gbuffer`
  - [`gbuffer_t *`](gbuffer_t)
  - The gbuffer to be removed and freed.
:::
        

---

**Return Value**


Returns `0` on success, or a negative value on error.
``````
