# modules/js

Placeholder for optional JavaScript modules. The active JS kernel and UI library now live under `kernel/js/`:

- `kernel/js/gobj-js/` — `@yuneta/gobj-js` (core GObj framework in JS)
- `kernel/js/lib-yui/` — `@yuneta/lib-yui` (reusable UI components)

This directory is kept for future JS-side modules that don't belong in the kernel.
