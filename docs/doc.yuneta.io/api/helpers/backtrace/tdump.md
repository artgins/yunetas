(tdump())=
# `tdump()`
<!-- ============================================================== -->


Dump the contents of memory in a human-readable format.
        

<!------------------------------------------------------------>
<!--                    Prototypes                          -->
<!------------------------------------------------------------>

``````{tab-set}

`````{tab-item} C

<!--====================================================-->
<!--                    Tab C                           -->
<!--====================================================-->

**Prototype**

```C

PUBLIC void tdump(
    const void  *data,
    size_t       size
);
        

```

**Parameters**


:::{list-table}
:widths: 20 20 60
:header-rows: 1
* - Key
  - Type
  - Description

* - `data`
  - `const void *`
  - The memory block to dump.

* - `size`
  - `size_t`
  - The size of the memory block in bytes.
:::
        

---

**Return Value**


No return value. This function outputs the memory dump to the standard output or log.
``````
