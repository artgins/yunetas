(istream_create())=
# `istream_create()`
<!-- ============================================================== -->


Create an input stream for reading data. Works with [`istream_h`](istream_h).

<!------------------------------------------------------------>
<!--                    Prototypes                          -->
<!------------------------------------------------------------>

``````{tab-set}

`````{tab-item} C

<!--====================================================-->
<!--                    Tab C                           -->
<!--====================================================-->

**Prototype**

```C

PUBLIC istream_t *istream_create(
    size_t buffer_size
);
        

```

**Parameters**


:::{list-table}
:widths: 10 5 40
:header-rows: 1
* - Key
  - Type
  - Description

* - `buffer_size`
  - `size_t`
  - The size of the buffer to allocate for the input stream.
:::
        

---

**Return Value**


Returns a pointer to the newly created [`istream_h`](istream_h), or `NULL` on failure.
``````
