# `node_uuid()`
<!-- ============================================================== -->


The `node_uuid` function retrieves the universally unique identifier (UUID) of the machine on which the code is running. This UUID can be used for identifying the machine in distributed systems or for logging purposes.

**Return Value**


The function returns a pointer to a string representing the UUID of the machine. This string is statically allocated and should not be freed by the caller.


**Notes**


The UUID returned by `node_uuid` is generated randomly and is unique to the machine. It is suitable for use in scenarios where a unique identifier is required.
