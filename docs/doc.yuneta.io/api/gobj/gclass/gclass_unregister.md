(gclass_unregister())=
# `gclass_unregister()`
<!-- ============================================================== -->

Unregisters a `gclass`, freeing its allocated resources if no instances exist.

<!------------------------------------------------------------>
<!--                    Prototypes                          -->
<!------------------------------------------------------------>

``````{tab-set}

`````{tab-item} C

<!--====================================================-->
<!--                    Tab C                           -->
<!--====================================================-->

**Prototype**

```C
void gclass_unregister(hgclass gclass);
```

**Parameters**

::: {list-table}
:widths: 20 20 60
:header-rows: 1

* - Key
  - Type
  - Description

* - `gclass`
  - `hgclass`
  - Handle to the `gclass` to be unregistered.
:::

---

**Return Value**

None.

**Notes**

If instances of the `gclass` still exist, an error is logged and the `gclass` is not unregistered.
``````
